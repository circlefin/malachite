# Exit Codes Reference

This document describes the exit codes used by Malachite components:

| Code | Meaning | Description |
|:---:|:---|:---|
| 0 | Success | Process completed successfully. |
| 1 | General Error | Catch-all for general errors. |
| 2 | Configuration Error | Missing or invalid configuration parameters. |
| 128 | Termination | Process terminated by an external signal. |
