# Malachite Glossary

Common terms used within the consensus mechanism:

- **Height**: The index of a block in the blockchain sequence. It increments with every finalized block.
- **Round**: A repeatable attempt to reach consensus for a specific Height. If a round fails, a new round starts at the same Height.
- **Step**: A distinct phase within a Round (e.g., Propose, Prevote, Precommit).
- **Validator**: A participant in the network responsible for proposing and voting on blocks.
